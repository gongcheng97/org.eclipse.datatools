************README***************
This Sybase ASE profile plugin (org.eclipse.datatools.enablement.sybase.ase) and 
it's associated models plugin (org.eclipse.datatools.enablement.sybase.ase.models)
replace the existing org.eclipse.datatools.enablement.sybase.ase.profile and 
org.eclipse.datatools.enablement.ase.dbdefinition plug-ins currently in
DTP Enablement. I recommend removing or disabling those plug-ins when you use
the new plug-ins.
